# HPM.INDEX

The simple index for [hpm](https://github.com/NCPlayz/hpm).

## General Format:

Currently, this is to implement typings for libraries.

```json
{
    "example": {
        "parent_package": "example",
        "url": {
            "type": "github",
            "link": "https://github.com/example/example"
        }
    }
}
```

## Available URL Types

- `github`
